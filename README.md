###	Description

NetPractice is a networking training project focused on understanding the fundamentals of IP addressing and routing. The goal of this project is to learn how network communication works by manually configuring IP addresses, subnet masks, and routes in different scenarios.

#	Instruction

To practice NetPractice locally, you need to download the file provided. Then, extract the archive inside a folder using the tar command like this:
-	tar -xf name_file
After that, simply execute the run.sh script. This will automatically open the index.html file in your default web browser, allowing you to start practicing.
When you complete a level, you can export your configuration using the "get my config" button located in the top left corner.
